# MERN Stack Shopping Cart App

https://maxmma.herokuapp.com/

Built with React/Redux, Express and MongoDB.

Functionality:

● User sign up and login.

● User profile page.

● Order details.

● Add item to shopping cart.

● Place an order.

● Checkout process(shipping method, payment method(PayPal)).
